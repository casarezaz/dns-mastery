================================================================================
USAGE EXAMPLES
================================================================================

Analyze configuration:
  python3 dns_threat_model.py analyze sample_data/before_architecture.json

Compare before/after:
  python3 dns_threat_model.py compare \
    sample_data/before_architecture.json \
    sample_data/after_architecture.json

Generate diagrams:
  python3 dns_threat_model.py diagram sample_data/before_architecture.json
  python3 dns_threat_model.py diagram sample_data/after_architecture.json

Reconstruct timeline:
  python3 dns_threat_model.py timeline sample_data/incident_events.json

JSON output:
  python3 dns_threat_model.py --format json analyze config.json

Run tests:
  python3 -m unittest test_threat_model -v
