# Lab 03: Query Path Tracer

**Phase:** DNS Foundations | **Week:** 3

> Resolution path tracing with latency measurement and anomaly detection.

---

## Overview

Traces the complete resolution path of a DNS query, measuring latency at each hop, identifying caching behavior, and detecting anomalies (unexpected resolvers, redirected queries).

## Tools

Python, dig, tcpdump/Wireshark

## Deliverable

Trace visualization + anomaly detection logic

## Reading

DNS and BIND — Chapter 2 (How DNS Works)
