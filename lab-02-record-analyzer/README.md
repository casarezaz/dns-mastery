# Lab 02: Record Type Analyzer

**Phase:** DNS Foundations | **Week:** 2

> Comprehensive DNS enumeration and security posture assessment tool.

---

## Overview

Performs full record-type enumeration of a target domain (A, AAAA, CNAME, MX, NS, PTR, SOA, TXT, SRV), identifies misconfigurations (missing SPF, no DMARC, open zone transfers), and generates a security posture report.

## Tools

Python (dnspython library), Bash

## Deliverable

Security report template + scanner script

## Reading

DNS and BIND — Chapters 4–5 (Resource Records, DNS Messages)
