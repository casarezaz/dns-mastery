# Lab 04: Zone Transfer Auditor

**Phase:** DNS Foundations | **Week:** 4

> AXFR misconfiguration scanner with sensitive record identification.

---

## Overview

Scans a list of domains for misconfigured nameservers that allow unauthorized zone transfers (AXFR). Parses transferred zone data to identify sensitive internal records. Ethical/authorized targets only.

## Tools

Python, dig, nmap (dns-zone-transfer script)

## Deliverable

Audit report template + scanning script + SIEM detection rule

## Reading

DNS and BIND — Chapter 6 (Zone Transfers)
