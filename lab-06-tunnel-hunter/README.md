# Lab 06: DNS Tunnel Hunter

**Phase:** DNS Security & Attack Techniques | **Week:** 6

> DNS tunneling detection engine tested against real tunneling tools.

---

## Overview

Detects DNS tunneling by analyzing query characteristics: subdomain length, entropy, query frequency, record type distribution, payload-to-query ratio. Tested against iodine, dnscat2, and dns2tcp traffic samples.

## Tools

Python, Wireshark, iodine/dnscat2 (lab environment only)

## Deliverable

Detection engine + Splunk dashboard + Sigma rules

## Reading

The Hidden Potential of DNS in Security — Tunneling chapters
