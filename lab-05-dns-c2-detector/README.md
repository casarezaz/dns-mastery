# Lab 05: DNS C2 Detector

**Phase:** DNS Security & Attack Techniques | **Week:** 5

> Detection logic for DNS-based command-and-control communication patterns.

---

## Overview

Using the CICIDS2017 dataset, builds detection logic that identifies DNS-based C2: periodic beaconing, unusual query volumes to single domains, high entropy subdomain queries, abnormal TXT record responses. Produces Splunk SPL and Sigma rules.

## Tools

Python, Splunk, Sigma, CICIDS2017 dataset

## Deliverable

Detection rules (SPL + Sigma) + analysis notebook

## Reading

The Hidden Potential of DNS in Security — C2 chapters
