# Lab 09: Encrypted DNS Analyzer

**Phase:** Modern DNS & Defensive Architecture | **Week:** 9

> Identifies DNS encryption protocols and detects DoH bypass attempts.

---

## Overview

Identifies which DNS encryption protocol a device/network is using (or if none), detects DoH bypass attempts, and validates certificate chains for DoT/DoH connections.

## Tools

Python, tshark/Wireshark, curl

## Deliverable

Detection script + DoH/DoT/DoQ comparison matrix

## Reading

The Hidden Potential of DNS in Security — Ch. 8-10 (Encrypted DNS)
