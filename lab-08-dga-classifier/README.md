# Lab 08: DGA Domain Classifier

**Phase:** DNS Security & Attack Techniques | **Week:** 8

## Overview

## Tools

Python, scikit-learn or TensorFlow, Jupyter

## Deliverable

Trained model + evaluation metrics + real-time classification script
