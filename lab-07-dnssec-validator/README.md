# Lab 07: DNSSEC Validator

**Phase:** DNS Security & Attack Techniques | **Week:** 7

> Full DNSSEC chain-of-trust validation from root KSK to target RRSIG.

---

## Overview

Validates the complete DNSSEC chain of trust for any domain. Identifies domains with broken DNSSEC, expired signatures, or missing DS records.

## Tools

Python (dnspython), dig +dnssec, dnsviz

## Deliverable

Validation tool + DNSSEC adoption report across Fortune 100

## Reading

DNS and BIND — DNSSEC chapters + RFC 4033-4035
